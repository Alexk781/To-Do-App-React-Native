<!-- GETTING STARTED -->

### Prerequisites

Make sure you have node installed. https://nodejs.org/en

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

To install this project make a clone of the files. Git clone: (https://github.com/Alexk781/To-Do-App-React-Native.git). Next open the folder and right click on the "To" file and open a terminal at the "To" file location. After the terminal is open, in that terminal use the command react-native run-ios. Once it generates metro type and I and it should launch the app.

<!-- USAGE EXAMPLES -->

## Usage

Users will be able to keep track of all of their tasks
Users can create tasks by adding a title, description, due date, priority level, and other relevant details
The app works offline, allowing users to add, modify, and view tasks even without an internet connection. The changes will be synced once the device is online again
